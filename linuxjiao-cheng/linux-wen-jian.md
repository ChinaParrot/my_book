# 一、文件基本属性
Linux系统是一种典型的多用户系统，不同的用户处于不同的地位，拥有不同的权限。为了保护系统的安全性，Linux系统对不同的用户访问同一文件（包括目录文件）的权限做了不同的规定。在Linux中我们可以使用ll或者ls –l命令来显示一个文件的属性以及文件所属的用户和组，如：


```
[root@bogon ~]# ll 
-rw-r--r--. 1 root root 8835 Mar  9 22:44 install.log

```

权限对象：
属主：u
属组：g
其他人: o
所有人：a

权限类型：
读：r		4
写：w		2
执行: x		1

**设置权限---chmod**
